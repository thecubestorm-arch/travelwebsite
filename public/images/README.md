# Image Resources

This folder contains all images for the Trail of China website.

## Folder Structure

- `china-basics/` — Thumbnails and images for the China Basics section pages
- `destinations/` — Photos for city guides (Shanghai, Beijing, Chongqing, etc.)
- `backgrounds/` — Hero images and full-width background photos

## Recommended Format

- Format: **WebP** (best compression + quality)
- Thumbnails (card previews): max **800 × 500 px**
- Hero / background images: max **1920 × 1080 px**
- Keep file size under 200 KB for thumbnails, under 500 KB for heroes

## Naming Convention

Use lowercase, hyphen-separated names matching the URL slug:
- `china-basics/what-apps-to-use.webp`
- `destinations/shanghai-hero.webp`
- `backgrounds/homepage-hero.webp`
